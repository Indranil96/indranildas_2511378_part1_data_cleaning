# indranildas_2511378_part1_data_cleaning

## Project Overview

This project focuses on cleaning, validating, and preparing the provided sales dataset for analysis by applying data quality checks, business rules, calculated fields, and summary reporting. The objective was to improve data quality by handling missing values, removing duplicate records, validating dates, applying business rules, creating calculated columns, and generating summary reports for business analysis.

## Cleaning Steps Performed

* Preserved the original raw dataset.
* Created a separate cleaned dataset.
* Filled missing Region values with **Unknown**.
* Filled missing Ship Mode values with **Unknown**.
* Replaced valid missing Discount values with 0.
* Removed exact duplicate records.
* Flagged negative discount values.
* Validated date sequence between Order Date and Ship Date.
* Standardized date formats.
* Created calculated columns:

  * calculated_sales
  * calculated_profit
  * profit_margin
  * shipping_delay_days
  * order_month
  * order_year
  * data_quality_flag

---

## Business Rules Applied

* Missing Region → Unknown
* Missing Ship Mode → Unknown
* Missing Discount → 0 only when related sales fields were valid
* Negative discounts flagged
* Exact duplicate records removed
* Ship Date earlier than Order Date flagged
* Calculated columns created for business analysis

* Missing Ship Mode records: 21
* Missing Discount records: 18
* Exact duplicate records removed: 25
* Negative Discount values flagged: 15
* Invalid Ship Date before Order Date records: 45

## Pivot Summary

Two pivot summary reports were created:

- **Sales by Region** – summarizes total sales across all regions, including records classified as Unknown.
- **Profit by Category** – summarizes total profit generated by each product category to support profitability analysis.

---

## Key Business Insights

* South region generated the highest sales.
* Unknown regions indicate incomplete source data.
* Profit by Category helps identify the most profitable product categories.
* Data cleaning significantly improved reporting reliability.

---

## Assumptions and Limitations

* Missing discounts were replaced with 0 only when other sales fields were valid.
* Original sales and profit columns were preserved.
* Negative discount values were flagged rather than modified.
* Invalid records were flagged instead of being deleted where required by the business rules.
